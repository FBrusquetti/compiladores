1- El archivo fuente con el codigo json debera estar en el mismo directorio que el archivo traductor.c con el nombre fuente.txt
2- Para compilar en linux use el compilador gcc desde la consola: gcc traductor.c -o nombre_del_ejecutable.
3- asegurese de que el binario tenga permisos de ejecucion con : chmod +x ./nombre_del_ejecutable
4- Para ejecutar: ./nombre_del_ejecutable
5- Al ejecutar generara el archivo de salida con el nombre output.xml
6- Si se detecta algun error durante el parseo los mismos seran mostrados por consola con la descripcion del error
y numero de linea.
